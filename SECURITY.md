# Security Policy

## Supported Versions

This project is a portfolio implementation and is maintained on the latest `main` branch.

## Reporting a Vulnerability

If you discover a vulnerability in this repository, please do not open a public issue with exploit details. Instead, describe the issue privately through GitHub security reporting or direct outreach so it can be reviewed and remediated responsibly.

## Scope

This repository is designed to demonstrate approval-routing, SLA escalation, and audit-history patterns. It does not ship with production identity, persistence, or secret-management infrastructure.
